Rating Explanation

Our scoring system evaluates the similarity between the entered keyword and domain names based on the following criteria:

Score 1: Very Low Similarity
Score 2: Low Similarity
Score 3: Moderate Similarity
Score 4: High Similarity
Score 5: Very High Similarity

How We Calculate Ratings

The similarity score for a domain is determined using the following steps:

  1. Extract and Normalize: Both the keyword and domain name are converted to lowercase for accurate comparison.
  2. Similarity Metrics: We calculate multiple similarity measures:
    • Exact Match: Compares the entire domain with the keyword.
    • Partial Match: Checks for partial matches within the domain.
    • Base Domain Match: Focuses on the main part of the domain (e.g., "example" from "example.com").
    • Keyword Coverage: Determines how much of the keyword is present in the domain name.
  3. Length Penalty: Penalizes overly long domain names to give preference to concise names.
  4. Combine Metrics: We use a weighted formula to calculate a total score:
    Total Score = (Exact Match * 0.4) + (Partial Match * 0.3) + (Base Domain Match * 0.3) + (Keyword Coverage * 20) - Length Penalty
  5. Map to Ratings: The total score is mapped to a rating:
    • Score > 90: 5 (Very High Similarity)
    • Score > 75: 4 (High Similarity)
    • Score > 50: 3 (Moderate Similarity)
    • Score > 30: 2 (Low Similarity)
